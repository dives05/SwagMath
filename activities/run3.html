Run 3

Description

Run3 is a classic online platformer with hundreds of levels and many character options for users to enjoy.

Gameplay

In Run 3, you control a little alien who's goal is to run from one end of the tube to another, while discovering new areas and characters.

Difficulty

The levels get progressively harder, with some paths having one aspect of difficulty in common, molding their themes

Design and Controls

Run 3 can be player with wasd or arrow keys.
